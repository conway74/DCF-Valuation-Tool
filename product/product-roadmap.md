# Roadmap

Building on: yfinance, numpy-financial, Streamlit

## Sections

### 1. Core Model (The "Happy Path")
A simple input form (Revenue, Growth, WACC) that outputs a single "Fair Value" using standard DCF math.

### 2. Sensitivity & Visualization
Interactive sliders and a "What-If" matrix (e.g. WACC vs Growth rate) to visualize valuation sensitivity.

### 3. Educational Overlay
The "Teacher" layer with tooltips, clear explanations of terms, and dynamic text explaining *why* the value changed.
